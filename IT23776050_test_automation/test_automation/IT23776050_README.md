## How to Run the Project

### Prerequisites

* Python 3.12 installed
* Google Chrome browser

---

### Setup Instructions

1. Open terminal inside the project folder:



2. Install dependencies

```bash
pip install -r IT23776050_requirements.txt
playwright install
```

---

### Run the Automation Script

```bash
py -3.12 IT23776050_test_automation.py --excel "IT23776050_Assignment 1 - Test cases.xlsx" --url "https://www.pixelssuite.com/chat-translator" --wait-ms 15000 --type-delay-ms 150 --slow-mo-ms 1000 --save-every 1 --keep-open
```


## Notes

* The script automates Singlish inputs into the Chat Translator and captures the output.
* Results are automatically updated in the Excel file.
* `--keep-open` keeps the browser open after execution for verification.
* Adjust wait and delay parameters if the website responds slowly.
